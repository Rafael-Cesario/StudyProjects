## Sobre o projeto

Este projeto foi criado para praticar autenticação de ponta a ponta. A proposta é entender, de forma direta, como funcionam cadastro, login, validação, hashing de senha, geração de token, persistência de sessão e integração entre front e back.  
A aplicação tem duas telas principais: criação de usuário e login. Elas se conectam a uma API que trata toda a lógica de segurança, validação e acesso aos dados.

## Tecnologias

**Front-end:** TypeScript, Next.js, React, Tailwind CSS, React Query, Next.js Cookies  
**Back-end:** TypeScript, Express, REST API, Zod, Bcrypt, JWT, Prisma, PostgreSQL, Docker  

## O que o projeto faz

- Cadastro de usuário com validação.  
- Login com senha criptografada.  
- Geração e verificação de JWT.  
- Armazenamento de sessão via cookies.  
- Comunicação entre front e back via REST.  
- Persistência dos dados em Postgres rodando em Docker.
